The Concept

This was a little side project to test out some illustrative work, and help clear the head between working on different projects and research. I first began drawing out the frame and forks, this acted as the base to work from. I then progress onto the wheels. When onto the more detailed bits I used reference images to help break the tricky components down into more simplistic shapes. The hardest art of this project was the drivetrain of the bike, in order to achieve this I had to illustrate it larger scale

Final Outcome

Overall I'm pretty happy with the final outcome of this project. I plan on returning to this project/illustrative bike concept again in the future. I think next time I will focus more on implementing detail into the components and sizing them accurately. This was defiantly the trickiest task when illustrating the bike. I also think it would be advantageous to create the components separate, allowing them to e used on different frame-set designs.
